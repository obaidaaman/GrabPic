# App Icons

Place your app icons here:

- `Icon-192.png` - 192x192 pixels
- `Icon-512.png` - 512x512 pixels

These icons are used for:
- PWA manifest
- Home screen icons on mobile devices
- Browser tab favicon

You can generate icons using tools like:
- https://realfavicongenerator.net/
- https://appicon.co/
